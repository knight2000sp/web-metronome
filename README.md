# web-metronome
Webメトロノーム web app

機能概要
- テンポ変更機能：スライダーでテンポを変更できる
- 拍子機能：1~9拍子を設定し、先頭にアクセント音を鳴らすことができる
- 分割機能：1拍子を1~6分割して、先頭にアクセント音を鳴らすことができる（3連符などを再現するイメージ）
- 音声操作機能：スタート・ストップ・テンポ・拍子・分割を音声で操作できる
- 注）拍子機能と分割機能は排他

音声操作詳細
- 【音声コントロール】をタップすると音声入力がONになる。Permissionは必要。
- 【音声停止】をタップすると音声入力がOFFになる（必要ない時にはOFFにしておくこと推奨）
- 音声入力がONの間、以下の音声コマンドを使用することができる「　」内に記した文言が音声コマンド。いずれも画面上のUIからも操作できる。
  - 開始・終了：「スタート」「ストップ」でメトロノームを開始・終了
  - BPM設定： 数値を話しかけると、BPMが変化する（例：「ひゃくにじゅう」と話しかけると、120BPMに設定される）
  - 拍子設定：「拍子」と話しかけると、画面に【拍子(N)の数値をどうぞ（3秒以内）】と表示されるので、その表示が見えている間に 1~9 の数値を話すと拍子が設定される
  - 分割設定：「分割」と話しかけると、画面に【分割(M)の数値をどうぞ（3秒以内）】と表示されるので、その表示が見えている間に 1~6 の数値を話すと分割が設定される
  - アクセントなし：「アクセントなし」と話しかけると、拍子・分割設定がOFFになる

制限事項（v0.7.1現在）
- 音声コントロールは、画面がOFFになると、以降効かなくなります。お手数ですがWebページをリロードして下さい（下スワイプでリロードされます）。
- 拍子や分割の数値を音声で入力する際、一桁の2を表す「に」 だけやたら認識率が低いです。「ゼロ二」 (02と認識され、2に変換される）や 「ふたつ」 など、別の言葉で置き換えてみてください。
